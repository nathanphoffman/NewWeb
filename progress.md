# Progress
As of June 26th, 2026 New Web has all of the major features needed for release.

Current experimental, but working, features are:
- Execution of WASM scripts
- Field Entry
- Markdown Editing
- Electron Browser

## Roadmap
The roadmap for the future is mainly building out these features, supporting them, and updating documentation.  At some point they will get a larger official release (like the Electron Browser) rather than requiring users to build it themselves.
