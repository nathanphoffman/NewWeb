# Creating your First Site
To get started with new web, you will need node/npm.  You can download them.

Then run the following commands in a terminal, replacing my-site with the name of your project (or leaving it my-site)
```
npx degit https://github.com/nathanphoffman/NewWeb/setup my-site
cd my-site
npm start
```

The website will tell you what port it started on, typically 8080, then open a browser and go to localhost:8080, you will see the hello world test page.

Now you can add and edit markdown files and slowly build out your website!
