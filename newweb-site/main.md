# Hello2 from NewWeb

This page is served by the **newweb-site** Bun server and rendered live via Go WASM + goldmark 222.  Woo woo Choo choo

## Editing this page

1. Log in via **Settings → Login** (`admin` / `password123` by default)
2. Click **Edit** in the nav bar
3. Make changes in the CodeMirror editor
4. Click **Save** — the file writes back to `newweb-site/main.md`

## Adding a page

Click **Add** in the nav bar after logging in. Enter a path like `about.md` and write your content.

Navigate to it with `#about` in the URL.
